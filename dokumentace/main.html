Cíl projektu

Cílem tohoto projektu bylo vytvořit jednoduchý textový editor pro tvorbu poznámek. Editor umožňuje vkládání a tvorbu obrázků a vkládání a náhrávání audia. Vytvořený dokument je možné uložit a načíst z disku počítače. Všechny audio prvky a obrázky jsou navíc na disku počítače k dispozici v mp3 respektive png formátu.

Popis funkčnosti

Základní editor je realizován pomocí atributu contenteditable. Nastavení tohoto atributu umožní uživateli interaktivně editovat text html prvku. 
Základní funkce pro formátování textu jsou realizovány pomocí funkce document.execCommand (tučný text, kurzíva, podtržení, zarovnání, odsazení, tlačítko zpět - díky tomu funguje jen na úpravy textu, vložení seznamu, oddělovací čáry a zrušení formátování).

Uložení a načtení dokumentu

Pro uložení dokumentu jsem použil File System Access API. Po stisku Ctrl+S (nebo prvním pokusu o vložení audia nebo obrázku) je uživatel vyzván k výběru lokace, kde chce dokument uložit (showDirectoryPicker). Následně je obsah dokumentu uložen v html formátu. Média jsou navíc ukládána na disk do stejné složky v mp3/png.
Dokument je možné z disku načíst po otevření sidebaru a stisku "Load document". Nebo je možné otevřit nový dokument pomocí "Create new document".

Tvorba obrázku

Uživatel může vytvářet obrázky pomocí canvas. Do canvas je možné nahrát vlastní obrázek buďto pomocí drag and drop na kreslící plochu nebo formulářového pole nad kreslící plochou. Obrázek je následně nahrán, je k němu vytvořena url, pomocí drawImage vykreslen do canvas a uložen na disk počítače pomocí File System Access API.


Přidání audia

Uživatel může vložit vlastní audio soubor pomocí formulářového prvku ve vyskakovacím okně pro vložení audia (nebo drag and drop na audio prvek). Může ale také audio v aplikaci přímo nahrát pomocí mediaDevices.getUserMedia a MediaRecorderu. Výsledný audio soubor je uložen na disk počítače pomocí File System Access API.
